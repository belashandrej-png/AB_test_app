# Руководство по внесению вклада (Contributing)

Благодарим за интерес к проекту **A/B Testing Academy**! Мы приветствуем любой вклад в развитие проекта.

##  Как можно помочь

-  Добавление новых терминов в глоссарий
-  Исправление ошибок
-  Предложения по улучшению
-  Улучшение дизайна
-  Дополнение материалов курса
-  Перевод на другие языки

##  Процесс внесения изменений

### 1. Fork репозитория

Нажмите кнопку **Fork** в правом верхнем углу страницы репозитория
